# 🔐 AI-Powered Phishing & Deepfake Audio Detection System

## 📌 Overview

The **AI-Powered Phishing & Deepfake Audio Detection System** is a final-year major project that integrates Artificial Intelligence (AI), Machine Learning (ML), Deep Learning (DL) and Cyber Security principles to detect modern cyber threats.

The system identifies:

- 📧 Phishing Emails, URLs, and Text Messages  
- 🎙️ AI-Generated / Spoofed (Deepfake) Audio  

With the rapid rise of social engineering attacks, phishing campaigns, and voice cloning scams, this project provides a modular and scalable cybersecurity solution capable of real-time detection and risk assessment.

---

## 🎯 Objectives

- Detect phishing attacks using NLP and ML models  
- Identify deepfake or spoofed audio using signal processing and deep learning  
- Integrate cybersecurity threat analysis techniques  
- Provide risk scoring for detected threats  
- Build a secure and scalable detection framework  

---

## 🛡️ Cyber Security Integration

This project incorporates cybersecurity mechanisms to improve detection reliability:

- 🔎 Threat Intelligence & Phishing Pattern Analysis  
- 🌐 URL Structure & Domain Feature Inspection  
- 📩 Email Metadata Awareness (SPF/DKIM Concepts)  
- 🔐 Secure API Handling & Input Validation  
- 📊 Risk Scoring System (Low / Medium / High Risk)  
- 📝 Logging & Monitoring for Threat Tracking  
- 🧪 Cyber Attack Simulation & Security Testing

## 🏗️ System Architecture

### 1️⃣ Data Collection
- Phishing email and URL datasets  
- Genuine and deepfake audio datasets  

### 2️⃣ Data Preprocessing
- Text cleaning and tokenization  
- TF-IDF / Word2Vec vectorization  
- Audio preprocessing  
- Feature extraction (MFCC, Spectrograms)  
- URL feature engineering  

### 3️⃣ Model Training
- Machine Learning classifiers for phishing detection  
- Deep Learning models for deepfake audio detection  

### 4️⃣ Cyber Security Layer
- Input sanitization  
- Threat classification  
- Risk scoring mechanism  
- Secure logging  

### 5️⃣ Deployment
- REST API using Flask  
- Web Interface using Streamlit
